# md-js

![Github Action Status](https://github.com/hangxingliu/mdjs/actions/workflows/v2.yaml/badge.svg?branch=master)

A lightweight Markdown parser

## Installation

``` bash
npm install md-js@2
```

##  Usage

``` javascript
const mdjs = require('md-js')
const html = mdjs.md2html("# Markdown content");
console.log(html);
```
